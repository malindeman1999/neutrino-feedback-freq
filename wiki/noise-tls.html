TLS Noise Vector

TLS resonance-frequency noise represented in the phase-like source direction.

Physical Expression

Common fractional frequency-noise form:

\[S_{TLS}(f)=A_{TLS}f^{-\beta}.\]

In the Lindeman mapping, TLS is treated as an effective fluctuating resonance frequency, aligned with the phase-like direction.

Design Scaling and Geometry Controls

For this project, TLS is parameterized directly by phase ASD at 100 Hz offset and a spectral exponent:

\[ \sqrt{S_{\phi,\mathrm{TLS}}}(100\,\mathrm{Hz}),\qquad \beta_{\mathrm{TLS}}. \]

Offset-frequency scaling is:

\[ \sqrt{S_{\phi,\mathrm{TLS}}}(\nu)=\sqrt{S_{\phi,\mathrm{TLS}}}(100\,\mathrm{Hz})\left(\frac{\nu}{100\,\mathrm{Hz}}\right)^{-\beta_{\mathrm{TLS}}/2}. \]

Then convert to fractional-frequency noise using the local phase slope:

\[ \left(\frac{S_f}{f_0^2}\right)_{\mathrm{TLS}}(\nu)= \left(\frac{\sqrt{S_{\phi,\mathrm{TLS}}}(\nu)}{f_0\left|d\phi/df\right|_{\delta f}}\right)^2. \]

IDC design intent is to reduce \(F\) and local electric field. For fixed stored energy \(U\),

\[ U=\frac{1}{2}CV^2\Rightarrow V=\sqrt{\frac{2U}{C}}, \]

so larger capacitance lowers field scale and typically reduces TLS coupling.

Lower \(f_0\) often helps indirectly because it usually implies larger \(C\) for fixed \(L\):

\[ f_0=\frac{1}{2\pi\sqrt{LC}},\qquad V\propto C^{-1/2}. \]

For first-pass quantitative estimates, see TLS Estimate. Geometry guidance is summarized in IDC Geometry.

Noise Vector Derivation

The active model treats TLS as its own phase-like source, independent of Johnson noise and independent of feedback gain. The source amplitude is calibrated against a passive microwave reference resonator with the same \(f_0\), \(Q_r\), effective \(Q_i\), and detuning, but with no electrothermal response and no heater feedback:

This means the input parameter \(\sqrt{S_{\phi,\mathrm{TLS}}}(100\,\mathrm{Hz})\) is interpreted as the measured Q/phase ASD that would be observed from TLS in the passive reference resonator, not as the raw right-hand-side matrix source amplitude.

\[ H_{\mathrm{TLS,passive}}(\nu)= \left[M_{\mathrm{passive}}^{-1}(\nu) \begin{bmatrix}0\\ i\end{bmatrix} \right]_{\phi}. \]

The target TLS phase ASD is

\[ \sqrt{S_{\phi,\mathrm{TLS}}(\nu)}= \sqrt{S_{\phi,\mathrm{TLS}}}(100\,\mathrm{Hz}) \left(\frac{\nu}{100\,\mathrm{Hz}}\right)^{-\beta_{\mathrm{TLS}}/2}. \]

The IQ source amplitude is chosen so the passive resonator would produce that target phase ASD:

\[ \sqrt{S_{N,\mathrm{TLS}}(\nu)}= \frac{\sqrt{S_{\phi,\mathrm{TLS}}(\nu)}}{|H_{\mathrm{TLS,passive}}(\nu)|}. \]

Equivalently, the USB/LSB TLS source is chosen once from the passive reference level, transformed into the model's normalized I/Q source coordinates, and then held fixed when the active electrothermal matrix is used.

The active-model TLS vector is therefore:

\[ N_{TLS}(f)= \begin{bmatrix} 0\\ i\sqrt{S_{N,\mathrm{TLS}}(f)}\\ 0\\ 0 \end{bmatrix}. \]

Using the project TLS parameterization:

\[ \left(\frac{S_f}{f_0^2}\right)_{\mathrm{TLS}}(\nu)= \left(\frac{ \sqrt{S_{\phi,\mathrm{TLS}}}(100\,\mathrm{Hz})\left(\dfrac{\nu}{100\,\mathrm{Hz}}\right)^{-\beta_{\mathrm{TLS}}/2} }{ f_0\left|d\phi/df\right|_{\delta f} }\right)^2. \]

This is a capacitor/frequency-noise interpretation only; the active source vector uses the passive-reference IQ normalization above.

Feedback Independence

The TLS source vector does not contain \(K_{\mathrm{fb}}\). Feedback only enters the active matrix \(M\), currently through the Q-column term \(m_{42}\). Thus:

\[ Y_{TLS}(\nu)=M^{-1}(\nu,K_{\mathrm{fb}})N_{TLS}(\nu), \qquad N_{TLS}(\nu)\ \mathrm{independent\ of}\ K_{\mathrm{fb}}. \]

For Q-channel NEP, this matches the cofactor argument on the feedback-invariance page: if source vectors are fixed and feedback appears only in the Q column, the \(K_{\mathrm{fb}}\)-dependent determinant cancels between noise transfer and signal responsivity.

Vector Component Units

\(N_{TLS}\) is also a mixed-unit \((r,\phi,T)\) vector, with component units:

\[ \mathrm{units}(N_{TLS})= \begin{bmatrix} 0\\ 1/\sqrt{\mathrm{Hz}}\\ 0\\ 0 \end{bmatrix}. \]

The \(\phi\)-component is an IQ-source ASD in \(1/\sqrt{\mathrm{Hz}}\), calibrated so the passive reference output equals the configured TLS phase ASD.

Use In Total Budget

Compute \\(Y_{TLS}=M^{-1}N_{TLS}\\), then combine in quadrature with other sources (and include correlations if measured).

Project criterion: in-band TLS phase noise should remain below the phonon-noise-equivalent floor over the analysis band.
