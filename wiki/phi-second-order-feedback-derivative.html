Second-Order Phi Feedback (Derivative Heater)

Duplicate of the phi-second-order feedback derivation, but with heater feedback proportional to \(\dot\phi\) instead of \(\phi\).

Home Project Theory Noise Sources Signal Processing Python Estimates Phi 2nd Order (Kp) Phi 2nd Order (Kd)

Starting Point

Use the same state \(y=[r,\phi,T_1,T_2]^T\) and linearized matrix structure as Phi 2nd Order, but replace proportional heater law with derivative heater law:

\[ P_{\rm heater}=K_d\,\dot\phi. \]

In frequency domain this is \(P_{\rm heater}=i\omega K_d\phi\). Therefore the row-4/col-2 matrix entry becomes

\[ m_{42}=Q_r x(\beta_A+2)P_2 - i\omega K_d \]

instead of \(Q_r x(\beta_A+2)P_2-K_{\rm fb}\).

Coupled \(\phi\)-\(T_2\) Equations

The row-2 phase equation is unchanged:

\[ \frac{2Q_i}{\omega_0}\dot\phi + \frac{4Q_iQ_r x}{\omega_0}\dot r - \beta_\phi r + \left(\frac{Q_i}{Q_r}+4Q_iQ_r x^2+2Q_r x\beta_\phi\right)\phi - \frac{2\alpha_\phi}{T_0}T_1 - \frac{2\alpha_{\phi2}}{T_{02}}T_2 =0. \]

The row-4 thermal equation becomes

\[ C_2\dot T_2 + \left(G_2-\frac{P_2\alpha_{A2}}{T_{02}}\right)T_2 + Q_r x(\beta_A+2)P_2\,\phi - K_d\dot\phi - \left(1+\frac{\beta_A}{2}\right)P_2 r =0. \]

Eliminated Second-Order Equation

Eliminating \(T_2\) again yields

\[ M_\phi\ddot\phi+\Gamma_\phi\dot\phi+K_\phi\phi=S_\phi(t), \]

with

\[ M_\phi=C_2\frac{2Q_i}{\omega_0}, \]

\[ \Gamma_\phi = C_2\left(\frac{Q_i}{Q_r}+4Q_iQ_r x^2+2Q_r x\beta_\phi\right) + \left(G_2-\frac{P_2\alpha_{A2}}{T_{02}}\right)\frac{2Q_i}{\omega_0} - \frac{2\alpha_{\phi2}}{T_{02}}K_d, \]

\[ K_\phi = \left(G_2-\frac{P_2\alpha_{A2}}{T_{02}}\right) \left(\frac{Q_i}{Q_r}+4Q_iQ_r x^2+2Q_r x\beta_\phi\right) + \frac{2\alpha_{\phi2}}{T_{02}}Q_r x(\beta_A+2)P_2. \]

Compared to proportional heater feedback, derivative heater feedback shifts \(\Gamma_\phi\) (damping) rather than \(K_\phi\) (spring), under this sign convention.

Simplified Form for \(\beta_A=\beta_\phi=\alpha_{A2}=0\) and \(x=0\)

The coupled equations reduce to

\[ \frac{2Q_i}{\omega_0}\dot\phi + \frac{Q_i}{Q_r}\phi - \frac{2\alpha_\phi}{T_0}T_1 - \frac{2\alpha_{\phi2}}{T_{02}}T_2 =0, \]

\[ C_2\dot T_2 + G_2T_2 - K_d\dot\phi =0. \]

After eliminating \(T_2\):

\[ M_\phi\ddot\phi+\Gamma_\phi\dot\phi+K_\phi\phi=S_\phi(t), \]

\[ M_\phi=C_2\frac{2Q_i}{\omega_0}, \qquad \Gamma_\phi=C_2\frac{Q_i}{Q_r}+G_2\frac{2Q_i}{\omega_0}-\frac{2\alpha_{\phi2}}{T_{02}}K_d, \qquad K_\phi=G_2\frac{Q_i}{Q_r}. \]

So in this special case, \(K_d\) changes damping only, while spring constant comes from the plant term \(G_2Q_i/Q_r\).

Design Targets for Ho Pulses

For a target \(|\phi|_{\max}<1\) rad and return time constant \(<1\) ms, use the reduced second-order form and tune near critical damping:

\[ \Gamma_\phi^2\approx4M_\phi K_\phi, \qquad \tau_{\rm crit}=\frac{2M_\phi}{\Gamma_\phi}. \]

  • Increase effective stiffness \(K_\phi\) (plant + optional proportional path) to reduce displacement scale.
  • Set \(K_d\) to place \(\Gamma_\phi\) near critical damping and suppress ringing.
  • Validate with real electronics bandwidth/noise/delay because derivative paths are phase-margin sensitive.

Assumptions

  • Same linearization as the main page: only constant and \(i\omega\) terms in \(M(\omega)\).
  • No extra heater state is introduced here; heater law is modeled as ideal instantaneous \(P_{\rm heater}\propto\dot\phi\).
  • If you later include controller or driver poles, add those as extra dynamic states.
