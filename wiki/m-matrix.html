Core Equation

The normalized small-signal system is written as:

\[ M(\omega)\,Y(\omega)=N(\omega),\qquad Y(\omega)=M^{-1}(\omega)\,N(\omega). \]

\(Y=[r,\phi,T_1,T_2]^T\) are demodulated normalized states and \(N\) is the source vector.

Role of \(M(\omega)\)

\(M(\omega)\) is the dynamic coupling operator that includes resonator response, dual-island thermal response, and electrothermal coupling.

  • Diagonal terms set intrinsic response poles/scales.
  • Off-diagonal terms encode coupling between \(r,\phi,T_1,T_2\).
  • Detuning enters through model coefficients and changes phase slope/responsivity.

Every source/noise path is computed through this same operator via \(Y=M^{-1}N\).

Implementation Form

In code, \(M(\omega)\) is built directly from project parameters and evaluated at demodulated frequency \(f\). The current implementation uses a dual-KID state \([r,\phi,T_1,T_2]\) with optional fallback to the original single-KID thermal subspace.

Full \(M(\omega)\) Matrix Form Used in Code

Let \(\omega_0=2\pi f_{0,\mathrm{eff}}\), demodulated angular frequency \(\omega\), loaded \(Q\equiv Q_r\), and dimensionless detuning \(x\equiv \delta f/f_{0,\mathrm{eff}}\). The implementation keeps the Eq. (13)-style electrical block and extends the thermal block to two islands:

\[ M(\omega)= \begin{bmatrix} \dfrac{2i\omega Q_i}{\omega_0}+\dfrac{Q_i}{Q}+\beta_A+4Q_iQx^2 & -\dfrac{4i\omega Q_iQx}{\omega_0} & -2Qx\beta_A+\dfrac{2\alpha_{A1}}{T_{01}} & \dfrac{2\alpha_{A2}}{T_{02}}\\[1.0em] \dfrac{4i\omega Q_iQx}{\omega_0}-\beta_\phi & \dfrac{2i\omega Q_i}{\omega_0}+\dfrac{Q_i}{Q}+4Q_iQx^2+2Qx\beta_\phi & -\dfrac{2\alpha_{\phi1}}{T_{01}} & -\dfrac{2\alpha_{\phi2}}{T_{02}}\\[1.0em] -\left(1+\dfrac{\beta_A}{2}\right)P_1 & Qx(\beta_A+2)P_1 & i\omega C_1+G_1-\dfrac{P_1\alpha_{A1}}{T_{01}} & 0\\[1.0em] -\left(1+\dfrac{\beta_A}{2}\right)P_2 & Qx(\beta_A+2)P_2-K_p-i\omega K_d & 0 & i\omega C_2+G_2-\dfrac{P_2\alpha_{A2}}{T_{02}} \end{bmatrix}. \]

Here \(K_p\) is heater proportional gain (\(\mathrm{W/rad}\)) and \(K_d\) is heater derivative gain (\(\mathrm{W\cdot s/rad}\)).

Here \(P_1+P_2=P_0\), with \(P_j\) split by series loss (\(P_j\propto R_j\)).

This is the exact matrix structure currently implemented in Sensor.m_matrix().

Connection to \(M_t\) and Stability

Time-domain linear dynamics are obtained from \(M(\omega)\) using:

\[ D_0=M(0),\qquad D_1=-i\,\frac{\partial M}{\partial\omega},\qquad M_t=-D_1^{-1}D_0. \]

See Mt Stability for eigenvalue checks and stability criteria.

Scope

This matrix model is a small-signal linearization about an operating point. Large-signal nonlinear behavior requires a nonlinear time-domain model.
