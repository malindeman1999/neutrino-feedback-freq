Power Partitioning in Two-KID Series Model

How total dissipated resonator power is split between KID1 and KID2 and where that split enters the matrix and noise vectors.

User Controls (Plotter)

In two-KID mode, KID2 thermal/electrical strength is controlled by ratios to KID1 baseline values:

\[ C_2/C_1,\quad G_2/G_1,\quad L_2/L_1,\quad R_2/R_1. \]

Setting all four ratios to 1 gives equal nominal branch properties relative to KID1.

Event heating split is controlled independently by:

\[ f_{E,1}\equiv \texttt{event\_power\_fraction\_kid1},\qquad f_{E,2}=1-f_{E,1}. \]

\(f_{E,1}=1\) means all event power drives KID1 (legacy behavior). \(f_{E,1}=0.5\) gives equal event partition.

Python: event_power_fraction_kid1, event_power_fraction_kid1_clamped, event_power_fraction_kid2.

Definition

The model uses one resonator current through two series losses. Therefore branch dissipation is proportional to branch series resistance:

\[ P_1 = P_0\frac{R_1}{R_1+R_2},\qquad P_2 = P_0\frac{R_2}{R_1+R_2},\qquad P_1+P_2=P_0. \]

When \(R_2\to 0\): \(P_1\to P_0\), \(P_2\to 0\), which recovers single-KID behavior.

Python: Sensor.P1_W, Sensor.P2_W, Sensor.R1_series_Ohm, series_R2_Ohm.

Where It Enters \(M(\omega)\)

The thermal/electrothermal rows are partitioned by \(P_1\) and \(P_2\):

\[ m_{31}=-\left(1+\frac{\beta_A}{2}\right)P_1,\quad m_{32}=Qx(\beta_A+2)P_1,\quad m_{33}=i\omega C_1+G_1-\frac{P_1\alpha_{A1}}{T_{01}}, \]

\[ m_{41}=-\left(1+\frac{\beta_A}{2}\right)P_2,\quad m_{42}=Qx(\beta_A+2)P_2-K_{fb},\quad m_{44}=i\omega C_2+G_2-\frac{P_2\alpha_{A2}}{T_{02}}. \]

Python: Sensor.m_matrix().

Event-to-readout responsivity uses the event partitioned drive vector \([0,0,f_{E,1},f_{E,2}]^T\), so NEP/signal normalization follows the chosen partition.

Where It Enters Noise Vectors

Johnson and electronic source amplitudes are branch-specific and use each branch power and branch temperature:

\[ N_{J1}\propto\sqrt{\frac{16k_B T_1}{P_1}},\qquad N_{J2}\propto\sqrt{\frac{16k_B T_2}{P_2}}. \]

Thermal-power branch terms use:

\[ N_{J1,\mathrm{th}}\propto\sqrt{4k_B T_1 P_1},\qquad N_{J2,\mathrm{th}}\propto\sqrt{4k_B T_2 P_2}. \]

Electronic excess (QP-generation-like) sources multiply each branch Johnson vector by branch factor:

\[ M_{e1}=\sqrt{\frac{E_{qp}}{k_B T_1}},\qquad M_{e2}=\sqrt{\frac{E_{qp}}{k_B T_2}}. \]

Python: Sensor.nj1_scale, Sensor.nj2_scale, Sensor.nj1_thermal_scale, Sensor.nj2_thermal_scale, Sensor.me_electronic_1, Sensor.me_electronic_2, Sensor.n_electronic_A_1, Sensor.n_electronic_A_2, Sensor.n_electronic_phi_1, Sensor.n_electronic_phi_2.

Consistency Checks

  • If KID2 is inactive (\(R_2=0\), \(L_2=0\), \(\alpha_{2}=0\)): branch-2 terms go to zero and model reduces to single-KID behavior.
  • If KID1 and KID2 are identical: \(P_1\approx P_2\approx P_0/2\), and branch noise terms become symmetric before propagation through \(M^{-1}\).
  • Feedback gain \(K_{fb}\) changes dynamics through \(m_{42}\), but does not directly alter source-vector normalization.
