Amplifier and generator noise mapped into source-vector form.

Physical Expressions

Input-referred sideband voltages:

\[V_A=\sqrt{k_B T_A Z_0},\qquad V_G\approx |S_{21}|\sqrt{k_B T_G Z_0}.\]

These enter similarly to external series sideband voltage sources.

The active model includes amplifier noise through the parameter \(T_A\), defaulting to \(1.3\,\mathrm{K}\), with \(Z_0=50\,\Omega\) at the readout reference plane.

Noise Vector Derivation

For a source that is uncorrelated between USB and LSB and uncorrelated with thermal power, use:

\[n_{RO}=\begin{bmatrix}v_{USB}\\v_{LSB}^*\\0\end{bmatrix},\quad \langle|v_{USB}|^2\rangle=\langle|v_{LSB}|^2\rangle=V_A^2+V_G^2. \]

The code constructs the amplifier modes in the sideband representation first:

\[ n_{\mathrm{amp},A}^{SB}=\frac{V_A}{\sqrt{2}} \begin{bmatrix}1\\1\\0\end{bmatrix}, \qquad n_{\mathrm{amp},\phi}^{SB}=\frac{V_A}{\sqrt{2}} \begin{bmatrix}-1\\1\\0\end{bmatrix}. \]

Transform to normalized coordinates with the readout-plane version of the Eq. (12) demodulation/normalization transform:

\[N_{RO}=T_V^{-1}d^{-1}n_{RO}.\]

Equivalent explicit decomposition is:

\[ N_{RO,A}= \begin{bmatrix} \sigma_{RO,A}\\ 0\\ 0 \end{bmatrix}, \qquad N_{RO,\phi}= \begin{bmatrix} 0\\ i\sigma_{RO,\phi}\\ 0 \end{bmatrix}, \]

with \(\sigma_{RO,A}\) and \(\sigma_{RO,\phi}\) set from measured or modeled readout PSD levels after demodulation/normalization.

For the paper's matched-resonator estimate, the resonance-circle radius at the amplifier is

\[ r_V=\frac{\sqrt{P_0Z_0}}{2}. \]

Demodulation adds the two independent sidebands in quadrature, so the normalized amplifier-noise ASD in both amplitude and phase coordinates is

\[ \sigma_{\mathrm{amp}}=\frac{\sqrt{2}V_A}{r_V} =\sqrt{\frac{8k_BT_A}{P_0}}. \]

The implementation uses the SB-derived vectors as the active source. It also computes the closed-form vectors \(N_{\mathrm{amp},A}=[\sigma_{\mathrm{amp}},0,0]^T\) and \(N_{\mathrm{amp},\phi}=[0,i\sigma_{\mathrm{amp}},0]^T\), and raises an error if the SB transform and algebraic estimate disagree.

If readout noise is measured directly in I/Q, it can be added as \\(\mathbf{S}_{amp}(\omega)\\) instead of via explicit \\(N\\) vectors.

Implemented pipeline: readout-originated voltage noise starts in SB (USB/LSB) form, is transformed by the SB\(\rightarrow\)IQ normalization matrix. Because amplifier noise is added after the resonator/bolometer, it is not injected as a thermal-power source.

Feedback Treatment

With detuning feedback, the controller uses measured phase, not only the device phase:

\[ \phi_{\mathrm{meas}}=\phi_{\mathrm{dev}}+q_{\phi,\mathrm{amp}}, \qquad dx=H_x(\omega)\phi_{\mathrm{meas}}. \]

Therefore amplifier noise contributes in two ways:

  • direct additive readout noise in the measured \(r\) or \(\phi\) channel;
  • feedback-injected detuning noise when the phase component \(q_{\phi,\mathrm{amp}}\) drives \(dx\).

For a readout-noise realization \(q=[q_r,q_\phi]^T\), the plant motion is solved from the augmented feedback matrix using

\[ \begin{bmatrix} M & -B_x\\ -H_x C_\phi & 1 \end{bmatrix} \begin{bmatrix} Y_{\mathrm{dev}}\\ dx \end{bmatrix} = \begin{bmatrix} 0\\ 0\\ 0\\ H_x q_\phi \end{bmatrix}. \]

The measured output is then

\[ Y_{\mathrm{meas}}=Y_{\mathrm{dev}}+ \begin{bmatrix} q_r\\ q_\phi\\ 0 \end{bmatrix}. \]

If \(H_x=0\), amplifier noise reduces to direct additive measurement noise and produces no device-state or temperature response. At the DC limit of an active integrator, the code uses \(\phi_{\mathrm{dev}}+q_\phi=0\), which is the finite equation's infinite-loop-gain limit.

Vector Component Units

Sideband-source vector units:

\[ \mathrm{units}(n_{RO})= \begin{bmatrix} \mathrm{V}/\sqrt{\mathrm{Hz}}\\ \mathrm{V}/\sqrt{\mathrm{Hz}}\\ 0 \end{bmatrix}. \]

After normalization transform, component units follow the Johnson-style \(N\)-basis:

\[ \mathrm{units}(N_{RO,A})= \begin{bmatrix} 1/\sqrt{\mathrm{Hz}}\\ 1/\sqrt{\mathrm{Hz}}\\ 0 \end{bmatrix}, \qquad \mathrm{units}(N_{RO,\phi})= \begin{bmatrix} 1/\sqrt{\mathrm{Hz}}\\ 1/\sqrt{\mathrm{Hz}}\\ 0 \end{bmatrix}. \]

Integration Into Model

The model includes two independent amplifier modes, \(q_A=[\sigma_{\mathrm{amp}},0]^T\) and \(q_\phi=[0,i\sigma_{\mathrm{amp}}]^T\). Their measured output PSDs are added in quadrature with Johnson, phonon, TLS, and electronic noise.

Do not also add a separate measured amplifier PSD block unless this internal amplifier-noise path is disabled; otherwise the readout noise is double counted.
