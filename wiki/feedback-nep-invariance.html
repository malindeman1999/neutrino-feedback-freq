Feedback NEP Invariance

Why Q-channel NEP is independent of heater feedback when feedback enters only the Q column.

Home Theory M Matrix Feedback NEP Noise Sources Signal Processing

Setup

The model state is

\[ Y=\begin{bmatrix}r&\phi&T_1&T_2\end{bmatrix}^T, \qquad M(\omega)Y(\omega)=N(\omega). \]

The measured Q channel is the phase coordinate \(\phi=Y_2\). Event power is injected on island 1, so the signal source vector is proportional to the third source component:

\[ N_{\mathrm{sig}}=\begin{bmatrix}0&0&P_{\mathrm{event}}&0\end{bmatrix}^T. \]

In the current model heater feedback appears only in the Q column of the matrix, specifically in \(m_{42}\), through \(K_p\) and \(K_d\):

\[ m_{42}=Q_r x(\beta_A+2)P_2-K_p-i\omega K_d. \]

Adjugate Form

The solution is

\[ Y=M^{-1}N=\frac{\operatorname{adj}(M)}{\det M}N. \]

The Q output is therefore

\[ \phi=Y_2=\sum_j (M^{-1})_{2j}N_j =\frac{1}{\det M}\sum_j C_{j2}N_j, \]

where \(C_{j2}\) is the cofactor formed by deleting row \(j\) and column 2 of \(M\). Because column 2 is deleted, no cofactor \(C_{j2}\) contains any element of the Q column:

\[ m_{12},\;m_{22},\;m_{32},\;m_{42}. \]

Thus the cofactors that map source vectors into Q do not contain the feedback term in \(m_{42}\).

NEP Cancellation

The Q responsivity to event power on island 1 is

\[ \frac{d\phi}{dP_{\mathrm{event}}}=(M^{-1})_{23}=\frac{C_{32}}{\det M}. \]

A noise source \(N^{(a)}\) produces

\[ \phi_a=\frac{C_{12}N^{(a)}_1+C_{22}N^{(a)}_2+C_{32}N^{(a)}_3+C_{42}N^{(a)}_4}{\det M}. \]

Therefore the input-referred Q-channel NEP for that source is

\[ \mathrm{NEP}_{a,Q}= \frac{ \left|C_{12}N^{(a)}_1+C_{22}N^{(a)}_2+C_{32}N^{(a)}_3+C_{42}N^{(a)}_4\right| }{ |C_{32}| }. \]

The common determinant cancels. Since the cofactors \(C_{j2}\) do not contain the Q-column entries, this NEP expression is independent of \(K_p\) and \(K_d\) when the source vectors are fixed.

Total Noise

For independent noise sources, the same argument applies after summing powers:

\[ \mathrm{NEP}_{Q}^2= \frac{ \sum_a \left| C_{12}N^{(a)}_1+C_{22}N^{(a)}_2+C_{32}N^{(a)}_3+C_{42}N^{(a)}_4 \right|^2 }{ |C_{32}|^2 }. \]

Thus the Q-channel NEP spectrum and energy resolution derived from that spectrum should be invariant under changes in \(K_p\) and \(K_d\), provided the source vectors and integration procedure are held fixed.

Assumptions

  • The output channel is Q/phase, \(Y_2\).
  • The event signal is power injected into island 1, \(N_3\).
  • Feedback enters only through \(m_{42}\), which is in the Q column.
  • Noise source vectors do not depend on \(K_p\) or \(K_d\).
  • No additional feedback electronics, heater-current, or readout noise is injected through the feedback path.
  • The same fixed frequency grid and estimator are used when comparing feedback gains.

Failure Modes

If computed Q-channel NEP changes with \(K_p\) or \(K_d\), one of the assumptions above has been violated or the implementation is not preserving the cofactor cancellation. Common causes are:

  • A noise vector depends directly or indirectly on feedback gain.
  • Noise sources are combined coherently before propagation instead of independently in power.
  • TLS scaling or another model component is normalized to a feedback-dependent Johnson quantity.
  • The energy-resolution integration bandwidth changes with feedback-dependent eigenvalues.
  • The calculation is not using the same source vector for signal responsivity and NEP normalization.
