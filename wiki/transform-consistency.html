Transform Consistency and Physics Invariance

Why applying the same demodulation/normalization transform to vectors and operators preserves physical predictions.

Home Theory M Matrix Transform Summary Transform Consistency Noise Sources

Core Statement

Yes: the model applies transforms consistently, so physics is preserved under basis change.

Key point: the matrix is formed in the same transformed/normalized coordinate system as the noise vectors. The operator and source terms are not mixed across bases.

Let \(y\) be a sideband-domain state and define normalized demodulated coordinates \(Y=S\,y\), where \(S=T_V^{-1}d^{-1}\). Then a linear system in sideband form,

\[ A\,y=n, \]

maps to

\[ M\,Y=N,\qquad M=SAS^{-1},\qquad N=Sn. \]

Therefore \(Y=M^{-1}N=S\,A^{-1}n\), so observables are unchanged by coordinate choice when both operator and sources are transformed consistently.

How This Project Applies It

  • \(M\) is formed, defined, and solved in normalized demodulated coordinates \([r,\phi,T_1,T_2]\).
  • Microwave/readout-originated noise sources (Johnson, excess electronic, TLS, readout) are built in sideband representation and mapped by SB\(\rightarrow\)IQ normalization before propagation.
  • Thermal-band sources (phonon) are injected directly in thermal components of \(N\) because they are not USB/LSB voltage sidebands.
  • All sources are propagated with the same rule \(Y=M^{-1}N\).

Consistency Checks Implemented in Code

The code computes SB-derived vectors and compares them against paper closed forms, then raises an error on significant mismatch:

  • Johnson Eq. (18) closed-form vectors vs SB-derived vectors.
  • Excess electronic Eq. (19) closed-form vectors vs SB-derived vectors.
  • TLS constructed-form check vs SB-derived mapped vector at each frequency.

Single-vs-dual split equivalence is also checked through shared SB\(\rightarrow\)IQ normalization assumptions.

Code references: Sensor.sb_to_iq_normalization_matrix, Sensor._assert_noise_vector_agreement, Sensor.n_johnson_A_1, Sensor.n_electronic_A_1, Sensor.n_tls_phi_at_hz.

Implication

Differences between configurations (single vs dual, feedback on vs off, detuning changes) should be interpreted as model-physics differences in \(M\), source amplitudes, or operating point, not as coordinate artifacts from inconsistent normalization.
